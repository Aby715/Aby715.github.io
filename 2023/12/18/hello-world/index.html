环境配置

  • 安装node.js
  • 安装npm
  • npm安装hexo

命令过程

  1. 创建新文件夹
  2. hexo init
  3. hexo s 这一步
  4. github上创建一个新repo，名称是{用户名}.github.io
  5. 修改本地_config.yml文件
  1. npm install hexo-deployer-git
  2. hexo clean
  3. hexo d
  4. 网址访问{用户名}.github.io，查看blog
